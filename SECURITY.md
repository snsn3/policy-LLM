# Security

We take security seriously. If you discover a security vulnerability in this project, please report it responsibly.

**Do not** open a public issue for security-sensitive problems.

Instead, email [Stany.Nzobonimpa@enap.ca](mailto:Stany.Nzobonimpa@enap.ca) with a description of the issue and steps to reproduce. We will respond as soon as possible.

Please do not disclose the vulnerability publicly until we have had a chance to address it.
