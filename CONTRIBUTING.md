# Contributing to Policy LLM

Thank you for your interest in contributing. This project supports academic research on AI and policy; we welcome issues, documentation improvements, and small code fixes.

## How to contribute

- **Bug reports and suggestions:** Open an [issue](https://github.com/snsn3/policy-LLM/issues) with a clear description and steps to reproduce (for bugs).
- **Documentation:** Fixes to README, CONTRIBUTING, or code comments are welcome via pull requests.
- **Code:** For larger changes, please open an issue first to discuss scope and approach.

## Development setup

```bash
git clone https://github.com/snsn3/policy-LLM.git
cd policy-LLM
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
export OPENAI_API_KEY=your_key
python app.py
```

## Pull request guidelines

- Keep changes focused; one logical change per PR when possible.
- Ensure the app still runs and the CI workflow passes.
- Update README or docs if you change setup or usage.

## Code of conduct

Be respectful and constructive. This project is tied to academic work and an inclusive community.
